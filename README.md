# react-native-vision-camera

The Camera library that sees the vision.

## Installation

```sh
npm install react-native-vision-camera
```

## Usage

```js
import VisionCamera from "react-native-vision-camera";

// ...

const result = await VisionCamera.multiply(3, 7);
```
